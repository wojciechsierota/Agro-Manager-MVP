# Agro Manager MVP

A professional farm management system designed to track field operations, costs, and revenues, integrated with an AI Agronomist Advisor.

## Features
- **Field Management**: Track multiple fields with area data.
- **Operation Logging**: Record tasks, dates, and costs.
- **Financial Reports**: Automatic balance calculation and CSV export.
- **AI Advisor**: Smart agronomy tips based on your farm data.

## Tech Stack
- **Language**: Python 3.x
- **Database**: SQLite3
- **AI Engine**: Google Gemini API
- **Format**: CSV for data exports

## Getting Started
1. Clone the repository.
2. Install required dependencies: `pip install google-generativeai`.
3. Set up your API Key in `ai_advisor.py`.
4. Run the app: `python main.py`.
